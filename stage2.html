SIGALE

📦 SIGALE est le logiciel ERP (Enterprise Resource Planning ou Progiciel de Gestion Intégré) central utilisé par l'entreprise.

Qu'est-ce qu'un ERP ?
C'est un système d'information qui centralise et gère l'activité quotidienne de l'entreprise (achats, gestion des stocks, logistique, RH, etc.) autour d'une base de données unique.

🎯 Lien avec mes missions :
Dans mon contexte, c'est l'outil métier au cœur de mon support matériel :

  • Les PDA Scanners dialoguent avec l'ERP pour mettre à jour les stocks en temps réel lors des flashages.
  • Les imprimantes thermiques Zebra reçoivent les ordres de l'ERP pour éditer les étiquettes de traçabilité.
  • Les PC et écrans de supervision permettent aux collaborateurs et managers de piloter les flux logistiques sur SIGALE.

Password Manager

Bitwarden est un gestionnaire de mots de passe (password manager). Il permet de stocker, générer et remplir automatiquement tes mots de passe de manière sécurisée.

Au lieu de retenir 50 mots de passe, tu n’en gardes qu’un seul :
👉 le mot de passe maître

🛠️ À quoi ça sert concrètement ?
Avec Bitwarden, tu peux :

  • 🔑 Enregistrer tous tes mots de passe (sites web, applis)
  • ⚡ Remplir automatiquement les formulaires
  • 🔐 Générer des mots de passe forts et uniques
  • 💳 Stocker des cartes bancaires
  • 🪪 Enregistrer des notes sécurisées (documents, infos sensibles)
  • 👥 Partager des accès en sécurité (famille / équipe)

🔒 Sécurité : pourquoi c’est fiable ?
Voici les points importants :

  • 🧠 Chiffrement de bout en bout (end-to-end encryption) → Même Bitwarden ne peut pas voir tes données.
  • 🔑 Chiffrement AES-256 (standard militaire)
  • 🔐 Support de la double authentification (2FA)
  • 🌍 Open-source (le code est public et vérifiable)
  • ☁️ Synchronisation entre tous tes appareils

Cybereason EPP

Cybereason est une plateforme professionnelle de cybersécurité utilisée par les entreprises pour détecter et stopper les cyberattaques.

👉 Elle protège les ordinateurs, serveurs et réseaux contre : Virus, Ransomwares, Hackers, Attaques avancées (APT), Intrusions internes.

Comment ça marche ?

  • 1️⃣ Installation d’un agent : Un petit programme est installé sur chaque ordinateur de l’entreprise.
  • 2️⃣ Surveillance continue : L’agent analyse les processus, fichiers, activités réseau et comportements suspects.
  • 3️⃣ Intelligence artificielle : Cybereason utilise le machine learning, l'analyse comportementale et la corrélation d’événements.
  • 4️⃣ Détection d’attaques : Par exemple, si un employé clique sur un email piégé et qu'un logiciel malveillant s’installe et essaie de se propager, Cybereason détecte la chaîne complète de l’attaque.

🚨 Que fait la plateforme quand elle détecte une menace ?
Elle peut : 🛑 Bloquer le programme, 🔒 Isoler un ordinateur du réseau, 🧹 Supprimer le malware, 📊 Générer un rapport détaillé, 👨‍💻 Alerter l’équipe sécurité.

PowerShell

🛡️ C’est quoi un SOC en cybersécurité ?

🎯 SOC = Security Operations Center
Un SOC est une équipe spécialisée qui surveille et défend le système informatique d’une entreprise 24h/24.
👉 C’est le centre de surveillance contre les cyberattaques.

🧠 Que fait un SOC concrètement ?
Les analystes SOC :

  • 👀 Surveillent les alertes de sécurité
  • 🔎 Analysent les comportements suspects
  • 🚨 Détectent les attaques
  • 🛑 Isolent les machines infectées
  • 🧹 Coordonnent la réponse à incident
  • 📊 Rédigent des rapports

🔗 Lien direct avec Cybereason
🛠️ Cybereason est un outil utilisé par le SOC.
👉 Le SOC = les humains
👉 Cybereason = l’outil technologique

Exemple concret :

  1. Un malware s’installe sur un PC (ex: via PowerShell).
  2. Cybereason détecte un comportement anormal.
  3. Une alerte apparaît dans le tableau de bord.
  4. L’analyste SOC analyse l’attaque.
  5. Il décide d’isoler le poste et lance la remédiation.

Donc : 🔎 Cybereason détecte, 🧠 Le SOC décide, 🛑 Le SOC agit.

Microsoft 365

☁️ Microsoft 365 (anciennement Office 365) est une suite cloud de productivité et de collaboration. Elle combine applications Office, messagerie professionnelle, stockage cloud, sécurité et administration centralisée.

  • 📧 Outlook : Messagerie professionnelle et calendrier.
  • 💬 Teams : Messagerie instantanée, visioconférence, partage de fichiers.
  • 📂 SharePoint : Intranet et gestion documentaire collaborative.
  • ☁️ OneDrive : Stockage cloud individuel sécurisé.
  • 📄 Suite Office : Word, Excel, PowerPoint (web + desktop).

🏢 Fonctionnement global (SaaS) :
Hébergement des données dans les datacenters Microsoft, accès web/app, synchronisation multi-appareils, gestion via Microsoft Admin Center.

🔐 Sécurité globale Microsoft 365 :
Gestion des identités (Azure AD), MFA, RBAC, DLP, Audit logs, accès conditionnel et chiffrement des données.

CATO Networks

Cato Networks est une plateforme de sécurité et réseau en mode SASE (Secure Access Service Edge). Elle combine réseau WAN + cybersécurité dans le cloud.

🎯 Objectif : Remplacer les MPLS/VPN, sécuriser le trafic Internet, protéger les utilisateurs distants et simplifier l’architecture IT.

🌐 Fonctionnalités Réseau :

  • 🔹 SD-WAN : Connexion multi-sites, gestion des liens (fibre, 4G...).
  • 🔹 Backbone Cloud Privé : Réseau mondial optimisé.
  • 🔹 Accès distant sécurisé (ZTNA) : Remplace les VPN classiques.

🔐 Fonctionnalités Cybersécurité :

  • 🔹 FWaaS / SWG : Firewall cloud et filtrage web.
  • 🔹 CASB / IPS : Contrôle SaaS et prévention d'intrusions.
  • 🔹 Anti-malware & DLP : Protection avancée et anti-fuite de données.

Microsoft Exchange

📧 Exchange Online est le serveur de messagerie cloud intégré à Microsoft 365 (qui remplace les serveurs locaux).

Fonctionnalités principales :

Microsoft Outlook

Microsoft Outlook est le client de messagerie utilisé pour accéder à Microsoft Exchange ou Microsoft 365.

Connecté nativement à Exchange, il permet la gestion des e-mails, du calendrier, des contacts, et s'intègre avec tout l'écosystème entreprise :

  • Centralisation de la communication interne

Workday (ERP SIRH)

☁️ Workday est une plateforme cloud de gestion d'entreprise (Enterprise Management Cloud) spécialisée dans les Ressources Humaines (SIRH / HCM) et la gestion financière.

Contrairement à un ERP traditionnel, c'est un système unifié en mode SaaS qui centralise toutes les données collaborateurs. Ses fonctionnalités principales :

  • Gestion du Capital Humain (HCM) : Recrutement, intégration et suivi des talents.
  • Gestion administrative et paie : Centralisation des données, congés, absences et notes de frais.
  • Portail employé (Self-service Cloud) : Permet aux salariés de consulter leurs fiches de paie, poser leurs congés et gérer leurs informations personnelles en toute autonomie.

Application Omni
